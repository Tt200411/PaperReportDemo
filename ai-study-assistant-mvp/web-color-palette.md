# Web Color Palette

## Core

- `--bg-app: #0F1013`
- `--bg-page: #141414`
- `--bg-panel: #17191E`
- `--bg-elevated: #232327`
- `--bg-card: #F3EBE2`
- `--bg-card-soft: #FFFDF9`

## Brand

- `--accent: #D4916E`
- `--accent-soft: #E7C7B0`
- `--accent-alt: #86A17D`

## Text

- `--text-primary: #FFFFFF`
- `--text-secondary: #B8B0A8`
- `--text-dark: #181818`
- `--text-dark-secondary: #6C675F`

## Border

- `--border-dark: #3C3935`
- `--border-light: #D5CCC3`

## Recommended Usage

- 页面总背景: `--bg-app`
- 顶栏、侧栏、深色工作区: `--bg-panel`
- 深色输入框、深色功能块: `--bg-elevated`
- 文档区、结果卡片、浅色信息面板: `--bg-card`
- 更干净的正文阅读底: `--bg-card-soft`
- 主按钮、重点 CTA: `--accent`
- 标签、高亮、小面积强调: `--accent-soft`
- 成功、稳定、辅助状态: `--accent-alt`
- 深底上的主文字: `--text-primary`
- 深底上的说明文字: `--text-secondary`
- 浅底上的主文字: `--text-dark`
- 浅底上的辅助文字: `--text-dark-secondary`

```css
  --bg-elevated: #232327;

  --accent-soft: #E7C7B0;
  --accent-alt: #86A17D;

  --text-primary: #FFFFFF;
  --text-secondary: #B8B0A8;
  --text-dark: #181818;
  --text-dark-secondary: #6C675F;

  --border-dark: #3C3935;

}
```
